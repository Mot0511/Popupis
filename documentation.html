Before start

Before start using Popupis, you must require some things:

  • Popupis
  • JQuery

You can paste this lines between <head> tags for require JQuery:
<script type="text/javascript" src="https://code.jquery.com/jquery-3.6.1.min.js"></script>
And you must paste this line up of </html> tag for require Popupis
<script type="text/javascript" src="popupis.js"></script>

Eventually, you must have just about this structure of your code:

          <!DOCTYPE html>
          <html lang="en" dir="ltr">
            <head>
              <meta charset="utf-8">
              <title></title>
              <script type="text/javascript" src="https://code.jquery.com/jquery-3.6.1.min.js"></script>
            </head>
            <body>

            </body>
            <script type="text/javascript" src="popupis.js"></script>

          </html>

Now we can start use Popupis

How to use it

First of all you must create two <div> blocks in each other and give parent <div> unique id, thus:

          <body>
            <div id="yourID">
              <div>

              </div>
            </div>
          </body>

It is your one popup window

Now, you must register new popup window. Add a <script> tag and call popupisInit() function between <script> tags.
Write array with identifiers of your popup as paramener in function. These code you must write down of the require Popupis, thus

          <script type="text/javascript" src="popupis.js"></script>

          <script type="text/javascript">
            popupisInit(['yourID']);
          </script>

Next, lets add a button for all popup. You must add onClick attribtue to button.
As a parameter write one of the these functions from table and write id of your popup in function, thus

          <body>
            <div id="yourID">
              <div>

              </div>
            </div>
            <button type="button" name="button" onClick="popup1('yourID')">Click here</button>
          </body>
